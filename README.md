# MyMeteo
A script to get weather data from the closest Polish weather station to given city using Nominatim and IMGW-PIB APIs.

Usage:
  ./mymeteo.sh [OPTIONS] [CITY]

Options:
  -h    Print help and exit.
  -v    Verbose mode.

Example:
  ./mymeteo.sh "KÃ³rnik"

RC File saved at: ~/.mymeteorc
Cache saved at: ~/.cache/mymeteo/stations_cache.json
